Module 3 — "ParallelCS": AI-Native, Product-Centric Curriculum Track
Loophole: Courses lag the AI-2026 stack by 3–5 years; faculty teach pre-LLM CS; students who want to actually build are starved of structured product-oriented learning.
Solution: An opinionated, opinionatedly AI-native curriculum site — think First-Principles-of-DSA + System Design + Agent Engineering + Ship-a-Product — delivered as 12-week tracks, each ending with a publicly hosted product. Crucially, this is not another video-course platform; it is a knowledge-graph-routed playlist of the best existing free content (3Blue1Brown, MIT OCW, Andrej Karpathy's neural-nets-from-scratch series, Anthropic's "Building Effective Agents" guide) with original glue: project briefs, eval rubrics, and an LLM-graded code reviewer.
The architecture is a curriculum knowledge graph (Neo4j or NetworkX-on-Postgres) where each concept is a node, each project is a milestone, and the LLM agent (Claude/GPT-4o-mini for cost) acts as a curriculum tutor that recommends the next node based on what a student has completed. This is the same multi-agent education pattern Gartner reported had a 1,445% surge in enterprise inquiries from Q1 2024 to Q2 2025; here it's just applied to a single Indian university. 8allocate
Tech stack: Astro static site + MDX content, Postgres + pgvector for content embeddings, a CrewAI orchestration with three agents (Curriculum Curator, Project Designer, Code Reviewer), GitHub Classroom for project submission. Cost: ~$30/month for 200 active students.
Anonymous deployment: Standard pseudonymous GitHub org under MIT license. Critical move — publish as a "remix", not "original work": every track explicitly attributes its sources, which both reduces legal exposure and reframes the project as curation rather than authorship, making stylometry less useful.
Adoption: Run a free "30-day AI-builder challenge" on Discord at the start of each semester; the first cohort's deployed projects become the marketing for the next. Students whose CGPA suffers because faculty marks attendance can still ship publicly visible products that get them placed via Module 2.
Precedent: Stoa School, Pesto, Masai, Newton School — all are external platforms that filled a Tier‑3 vacuum. None had institutional permission. The difference here is that the brand never charges money; the product is course-aligned, so a Shoolini CSE student uses it during their semester, not after graduation.
Risks: Plagiarism complaints from curriculum owners — mitigated by linking, not copying. Faculty backlash that students using it are "distracted" — mitigated by openly aligning the project briefs with the official Shoolini syllabus topics, so the student can claim "I built this for my Operating Systems mini-project."
